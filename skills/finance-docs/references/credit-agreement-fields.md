# Credit Agreement Fields (v1)

- Parties: borrower, guarantor, lender, agent
- Facility: type, committed amount, accordion, currency
- Pricing: benchmark, spread, floor, default rate
- Tenor: signing date, effective date, maturity date
- Covenants: leverage, coverage, liquidity, reporting
- Default: payment, covenant, insolvency, cross-default
- Collateral: pledged assets, guarantees, perfection notes
